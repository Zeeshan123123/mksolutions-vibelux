## 💡 Sample Workflow

1. **Start Project**
   - Select "Cannabis Cultivation"
   - Enter facility size
   - Choose location for codes

2. **Design Layout**
   - Draw rooms or import CAD
   - Define cultivation zones
   - Set environmental targets

3. **Place Fixtures**
   - Search DLC database
   - Use auto-arrangement
   - Fine-tune placement
   - Check PPFD heatmap

4. **Engineering Systems**
   - Auto-generate electrical
   - Design HVAC zones
   - Add control systems
   - Verify compliance

5. **Optimize & Export**
   - Run AI optimization
   - Review suggestions
   - Generate documents
   - Export for permits
